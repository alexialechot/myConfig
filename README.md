# Brew commande
```
 brew search packages
 brew install packages
 brew cask install packages
 brew remove packages
 brew cask remove 
```
#### Verify if brew config is okay
```
brew doctor
```
#### Up to date Brew + cleanning
```
brew update; brew upgrade; brew cleanup; rm -rf (brew --cache)
```
